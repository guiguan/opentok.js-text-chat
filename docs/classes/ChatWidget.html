onMessageReadyToSend

(
  • contents
  • callback
)

Defined in /home/salva/workspace/opentok-textchat/src/ChatWidget.js:76

Called when the user clicks on the send button. It will received the contents from the input area and a callback to call once it finishes to send the message and it can be displayed in the conversation area.

Parameters:

  • contents String

    Contents of the input area at the moment the user clicked on the send button.

  • callback Function

    Function to call once the message is ready to be displayed in the conversation area (usually after it was successfully sent). Pass no parameters to denote success and some error to indicate a failure.
